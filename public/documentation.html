Projects

Get all users

GET /users?api_key=[API_KEY]

Result:

{
    "data": [
        {
            "user_id": 1,
            "email": "test@test.se"
        }
    ]
}

Get specific user

GET /users/1?api_key=[API_KEY]

Result:

{
    "data": {
        "user_id": 1,
        "email": "test@test.se"
    }
}
